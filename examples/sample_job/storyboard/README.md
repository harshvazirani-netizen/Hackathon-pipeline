# Storyboard frames go here

One approved reference image per beat, named in beat order:

```
beat_01.png   # the look for BEAT 1
beat_02.png   # the look for BEAT 2
beat_03.png   # the look for BEAT 3
```

- Sorted filename order = beat order, and must line up with the screenplay's beats.
- `.png` / `.jpg` / `.jpeg` / `.webp` all work.
- The pipeline animates **from** these frames (they ARE the approved look).

(This folder ships empty — drop your real frames in before running.)
